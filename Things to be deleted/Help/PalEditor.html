Palette Editor

The palette editor allows you to change the colors used for graphics for each level. Each level has a set of four palettes for the background and a set of four palettes for sprites. Each palette has four colors in it. Additionally, Brinstar and Norfair each have a secondary set of background palettes.

Each horizontal row of four squares represents a palette. To edit an entry in a palette, click it and then select a color in the color table below. When setting a palette entry to black, it is recommended that the black square marked with a white triangle be selected from the color table.

Care should be taken when editing sprite palettes. Samus, her weapons, doors, and her item display all share palettes with each other or enemies.
